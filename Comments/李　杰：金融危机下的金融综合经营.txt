<p><span class="t14">   <span class="t14"> 近日，金融业最轰动、最引人瞩目的事件，非平安保险集团收购深圳发展银行莫属。 一时间，“银保股权合作大戏上演”，“保险巨头加速金融控股梦”等醒目的标题占据了各大经济媒体的头版头条，似乎在为此次平安与深圳发展银行的联姻这一历史时刻提供鉴证。然而，回顾这幕经典“影片”的主角-平安集团的银行发展路线图，可以清晰地看到，自从1999年平安保险集团确定以打造“提供综合金融服务”的发展战略开始，它的“演出”就从来没有停止过。<br/>
<span class="t14">   <span class="t14"> 2003年底，中国平安与汇丰银行联合收购福建亚洲银行100%的股权，交易在2004年一季度完成，并最终以平安集团控制73%的股权告终，福建亚洲银行也因此改名为平安银行。当时的福建亚洲银行只是一家负债缠身，资产质量较差的地方性商业银行，业务范围也仅限于福州市。经过两年的发展，平安银行并没有实现质的突破，网点也只有上海总部和福州分公司两家。2006年7月，平安集团又以几乎相同的方式收购了深圳市商业银行。深圳商业银行是我国成立的第一家城市商业银行， 2006年深圳商业银行的资产规模在全国城市商业银行中占据第三的位置。平安集团将深圳商业银行的资产整合到平安银行中，实现了平安集团银行业发展的一次大跨越。虽然平安银行发展的脚步并没有停止，但是迄今为止仍然是一家只有7家分行的区域性银行。网点资源的缺乏一直是困扰平安银行发展的一大障碍，平安银行的发展规模显然无法满足平安集团“提供综合金融服务”的要求。<br/>
<span class="t14">   <span class="t14"> 不过，如果把平安银行前几年来的发展比作是学习阶段的话，那么终有学成出师的一天，如果把上述的收购和整合比作涓涓溪流的话，那么终有汇成大江大河的一天。从这一点来看，平安集团与深圳发展银行今天的联姻就显得既“合理”又“正常”。因为在网点规模上，深发展在全国18个中心城市拥有的200多家分支机构的网点；在品牌资源上，深发展有一张具有全国影响力的银行牌照、网络和客户资源。从资产规模来看，“中国平安在今年一季度的资产总额为7500亿元，深发展为5200亿元，两家公司联姻后资产规模达到1.27万亿元，在目前上市的资产规模超万亿的金融机构中排名第八。在银行业务上，平安银行和深发展相加也有6670亿元的规模，在目前上市的银行中排名第十。”<br/>
<span class="t14">   <span class="t14"> 但是，笔者认为单纯用“门当户对”，“郎才女貌”来完全肯定两家集团此次联姻的合理性还是有失偏颇，因为时间的特殊性，市场环境的特殊性仍然是分析这一收购事件需要重点考虑的因素。不要忘记当前我国正在经历经济发展的困难时期，也不要忘记全球金融危机的阴霾还没有完全消散。从这个角度来看，才能发现此次平安巨资收购深发展的特殊性。在此笔者不想再赘谈金融综合经营能够带来的收益，也不想再多提“资源整合是关键”、“财务健康是保证”、“保险业务是核心”等建议，因为纸上得来终觉浅，而且，作为收购的当事人在这方面应该比我们做了更加充分的分析和准备。笔者以为，总结此次平安与深发展的联姻给我国金融业未来发展带来的启示才更为重要，也更加有意义。<br/>
<span class="t14">   <span class="t14"> 对于此次收购行动，它给我们带来的重要意义和启示在于：<br/>
<span class="t14">    首先，这一次收购显示了我国金融企业的发展战略重点将集中于国内经济的发展。金融危机对平安的影响和打击是巨大的，投资富通银行的巨亏直接导致了平安集团2008年的净利润下降95%以上。而此次的收购行为则证明平安的发展战略已经发生了转移，那个曾经四处出击的“懵懂少年”也终于开始认识到把握住国内市场才是自身健康发展的前提。平安的战略调整为我国金融业稳健经营提供了宝贵的经验和教训，也为我国所有金融企业的发展指明了方向。<br/>
<span class="t14">    其次，此次收购还证明了在金融危机的背景下对金融综合经营前景的怀疑可能需要划上一个句号。平安收购深发展说明了我国金融业起码是保险业的综合经营冲动并没有停止，追求金融综合经营所带来的规模经济和范围经济仍然是保险业巨头们努力追求的战略方向。而且，可以预见，未来以平安、人寿为主的保险业巨头们将仍然会活跃在金融跨行业扩展的领域中。当然，对金融综合经营的争论也许会一直持续。但不可否认的是，金融综合经营是未来我们需要长期面对的现实问题，对于问题的正确态度不是回避，不是拖延，而是尽快拿出对策去防范和解决。<br/>
<span class="t14">   <span class="t14"> 最后，此次收购拉开了我国金融综合经营发展的新阶段。曾经我们对于金融综合经营关注的焦点一直局限在交叉销售，投资收益等低层次上，而忽视了金融综合经营中的产品资源整合这一核心的创新上。通过平安银行近年来的发展可以看到，当前平安已经在银行产品和保险产品的功能整合上迈出了一大步，如平安银行一直努力打造的平安信用卡已经发展成为集银行产品与保险产品功能（包括航意险、医疗险）于一身的综合性金融产品。而此次收购深发展不仅可以为这一综合性金融产品提供更大的市场空间，而且可以为不同金融产品的进一步整合提供更大的操作空间。这无疑将大大推动我国金融综合经营的发展。<br/>
<span class="t14">   <span class="t14"> 我们常说，危机是危险也是机遇，也许只有在金融危机中，我们才有机会拨开迷雾，正确判断未来金融业的发展方向；也许只有在金融危机下，我们才会发现金融综合经营原来的确是大势所趋。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p>
