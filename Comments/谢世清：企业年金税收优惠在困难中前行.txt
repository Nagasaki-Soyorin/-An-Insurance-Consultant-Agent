<p><span class="t14">  <span class="t14"> <span class="t14"> 今年6月20日，保监会与天津市政府共同发布了《天津滨海新区补充养老保险试点实施细则》。该《细则》在补充养老保险的个人税收优惠政策突破了原有框架，曾一度被保险业内人士普遍看好。但近日，由于国家税务总局对试点中个人购买补充养老险可税前列支30％薪金持有异议，所以这一企业年金税收优惠的试点方案被迫紧急叫停。<br/><span class="t14">  <span class="t14"> <span class="t14"> 这无疑给刚刚取得突破的企业年金税收优惠制度泼了一盆冷水。由于中国的企业年金制度刚刚起步，各方面配套制度也亟待完善，税收优惠制度的建立肯定不会是一蹴而就和一帆风顺的。但是个人税收优惠制度是世界各国企业年金得以发展的重要条件，也是我国企业年金发展的必由之路。<br/><span class="t14">  <span class="t14"> <span class="t14"> 《细则》规定，在天津滨海注册的企业和职工购买在补充养老保险时，可享受税收惠政策：企业为职工购买补充养老保险的费用支出在本企业上年度职工工资总额8％以内的部分，可在企业所得税前扣除；补充养老保险的个人交费部分，则为个人工资薪金收入30％以内的部分可在个人所得税前扣除。<br/><span class="t14">  <span class="t14"> 据报道，叫停的原因主要有三点：（1）个人税收优惠缺乏法律依据，与之前国家税务总局出台的政策相抵触；（2）30%的比例值得商榷，国税总局认为这一比例太高；（3）税收优惠方式没有明确说明，企业年金累积和领取阶段是否享受税收优惠在《细则》中没有明确规定。<br/><span class="t14">  <span class="t14"> <span class="t14"> 对于这三点，本文认为若想要推动企业年金在我国的发展，个人税收优惠政策的突破是不可避免的问题，需要各部门大力合作，协调一致，尽早解决。另一方面，就在此试点方案来看，30%的优惠幅度确实显得有些激进。根据国际经验，10%是一个较为恰当的标准。同时，应该明确在企业年金缴费、积累和领取三阶段采取EET税收优惠方式。具体意见如下：<br/><span class="t14">  <span class="t14"> <span class="t14"> 首先，企业年金税收优惠制度的发展离不开各个部门的努力，需要一整套协调一致的法律框架和政策。构建一个合适的法律环境，是企业年金发展的基础。一直以来我国关于企业年金税收优惠的政策规定显得模糊与混乱。这是一个无法回避和亟待解决的关键问题。显然，要通过年金税收优惠计划来推动补充养老保险这个我国养老保险第二支柱的发展，离不开税收部门理解、参与和配合。<br/><span class="t14">  <span class="t14"> <span class="t14"> 第二，比较国际上的实际现状，天津滨海新区对个人给予30%的税收优惠比例确实显得偏高和冒进。即使一些高福利国家也没有达到这个标准。例如福利水平很高的加拿大，其个人税收优惠比例也只有18%。从试点意义上讲，30%的比例缺乏示范效应。如此之高的税收优惠是很难保持可持续性，更难在更大的范围内推广。因此，一下子制定如此之高是税收优惠政策，就很难能被国家税务部门所理解、所接受。<br/><span class="t14">  <span class="t14"> <span class="t14"> 第三，不仅要有税收优惠的比例限制，而且要有额度限制。美国的401k计划在这方面给我们提供了一个有益的参照。美国401k计划在2000年时，《国内税收法》规定税前缴费最高限额为1.05万美元，总缴费（税前企业缴费+税前个人缴费+税后员工缴费）上限为年工资的25%或3万美元，两者之中取较小者为标准。401k计划所规定的税收优惠上限大概是平均个人年工资。这样可以防止高收入者过度使用税收优惠，加剧社会贫富差距。但是目前《细则》规定的只有38%（企业缴费优惠8%+个人缴费优惠30%）的比例，没有缴费的额度限制。30%的比例会为高收入人群提供合理的避税方式，使得他们才是真正的最大受益者，从而不利于和谐社会的形成。<br/><span class="t14">  <span class="t14"> <span class="t14"> 第四，不仅要科学合理地制定税收优惠的幅度，也要明确税收优惠的方式。企业年金的运行有三个环节：缴费环节、积累环节和领取环节。企业年金的税收优惠模式可以按照征收环节的不同进行分类。如果借助字母E（exempt，即免税）和T（tax，即征税）表示政府在上述环节的征税情况，那么企业年金税收政策主要有以下几个类型：?EET、ETT、TEE、TTE模式。<br/><span class="t14">  <span class="t14"> <span class="t14"> 国际上大部分采用EET模式。美、英、日等国一般对企业和职工缴纳年金费用以及年金基金投资免税，只对退休职工领取企业年金进行征税。在这种情况下，企业和职工都享受到税收优惠，有利于年金的发展。目前，《细则》只有在第一个环节上有税收优惠，而在积累和领取环节上是否有税收优惠，没有给予明确说明。在未来我国企业年金税收优惠改革实践中，应该克服这一缺陷，明确采取国际上的主流EET模式。<br/><span class="t14">  <span class="t14"> <span class="t14"> 第五，一套合理的税收优惠计划对于我国企业年金市场发展有重要意义。它可以鼓励企业和个人参与企业年金计划中来，逐步走出当前企业年金市场中“机构热、企业冷”的困境。企业年金的意义在于企业与劳动者自觉为以养老为目的在当前建立基金，为劳动者提供除社会保险之外的养老保障，同时也为企业提供了有效的员工激励机制。但是，如果没有针对个人的税收优惠，个人缺少足够的动力参与到企业年金中去，则这两个作用将会受到极大限制。<br/><span class="t14">  <span class="t14"> <span class="t14"> 总而言之，尽管天津滨海养老保险试点存在着缺乏相应的现有法规政策支持、个人税收优惠幅度偏大和优惠方式不明确等诸多不足，但是它毕竟在个人税收优惠政策上面进行了大胆的尝试。个人税收优惠政策改革无疑是我国企业年金发展的关键。尽管目前遇到挫折，但是个人税收优惠是企业年金政策改革的战略方向之一，也必将在困难中前行。
