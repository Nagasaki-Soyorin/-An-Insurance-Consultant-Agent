<p><span class="t14">  <span class="t14"> 5月26日，保监会网站照例公布了《2009年1季度保险中介市场发展报告》，一个小小的数字引起了笔者的兴趣：截至2009年3月31日，全国共有保险专业中介机构2439家。但是《二○○八年保险中介市场发展报告》数据显示：截至2008年年底，全国保险专业中介机构数量为2445家。这标志着2002年以来迅速增长的专业中介机构数量第一次出现了负增长。<br/>
<span class="t14">  <span class="t14"> 这个“转折点”的出现虽然有些“突然”，却也并不让人意外！一方面，虽然近两年来专业中介机构数量的增速放缓，但2007年新设机构与退出机构比例仍维持在为2：1（批设346家，174家终止经营），2008年机构数量净增加了114家，而2009年第1季度即出现负增长，此乃“突然”之处。另一方面，“入市”以来中介市场实行市场化准入制度，专业中介机构数量迅速增加，各个公司经营管理水平和资金实力参差不齐、整体专业化水平不高、市场竞争不规范等问题一直存在。这种背景下发展起来的专业中介市场在激烈的市场竞争过程中必然会经过一个优胜劣汰的“阵痛调整期”。“转折点”的迟早出现，自然也在预料之中。<br/>
<span class="t14">  <span class="t14"> 对待这样一个“转折点”，我们不应“小题大做”，却也不能不“严肃对待”！所谓不应“小题大做”，是指我们应当正确认识到“转折点”出现的合理性。这主要表现在：一是，专业中介市场的容量有限，机构数量的增长必然会遇到“瓶颈”；二是，经营失败的机构的退出，以及市场投资者的更加理性，是市场化机制的必然结果；三是，受金融危机的影响，使市场的优胜劣汰作用加速，从而“转折点”的提前出现也在合理之中；四是，退出机构的比例在市场可承受的范围之内（以代理公司为例，今年1季度共退出43家，约占代理公司总量的2.5%），不会对中介市场以及保险市场的稳定造成太大影响。<br/>
<span class="t14">  <span class="t14"> 对于中介市场过去十年的发展，我们应当更多的予以肯定。因为，在过去十年的发展中，无论是在机构数量、代理保费收入（或估损金额），还是经营收入、盈利状况方面，专业中介市场都取得了巨大的发展。2008年全国2445家保险专业中介机构的总资产达到94.94亿元，整体盈利25502.19万元；全国专业代理和经纪渠道实现的保费收入515.04亿元，占全国保费5.27%；保险公估公司评估估损金额达264.96亿元。这些简单的数字，不仅代表了专业中介市场的发展成就，同时也为其未来“质”的发展奠定了“量”的基础。<br/>
<span class="t14">  <span class="t14"> 但是，在“转折点”的警示作用之下，对于专业中介市场未来的发展方向，我们却不能不“严肃对待”。因为，在经过十来年“量”的初始积累之后， “质”的发展是关系到专业中介市场未来生存和发展的根本问题所在。那么，专业中介市场未来将向何处去？其生存和发展的“立业之本”又在哪呢？<br/>
<span class="t14">  <span class="t14"> 从本质上来看，专业中介市场的发展是保险公司的产品销售、售后服务、事故查勘估损等职能的逐步市场化过程，因而专业中介市场的发达程度往往成为衡量一国保险市场成熟与否的标志。新制度经济学的交易成本理论认为企业内部分工和外部市场交易是组织生产、资源配置的两种基本方式，究竟是选择企业机制还是市场机制，取决于两者的交易成本孰高孰低。也就是说，保险公司是选择将销售、服务、估损等职能内部化，还是交由专业中介机构完成，取决于两者成本或效率的高低。<br/>
<span class="t14">  <span class="t14"> 那么，如何才能使专业中介机构在与保险公司内部销售、服务和损失查勘部门的竞争中取得成本和效率上的优势，从而赢得长期生存和发展空间呢？其基本途径在于：或是依靠其庞大的网点优势和灵活的经营机制，降低保险公司的经营成本；或是依靠其“中间人”的地位，树立中介市场的诚信形象，为消费者提供优质高效的专业服务，赢得消费者的认可；或是凭借其敏锐的市场嗅觉，为保险公司的产品开发和销售服务创新提供及时的市场信息。总的来说，就是要凭借专业中介机构的“专业”优势，促进市场供需双方的有效沟通，提高保险市场整体的风险管理水平和经营管理效率。<br/>
<span class="t14">  <span class="t14"> 但是，目前我国的专业中介机构大多是依靠股东和关系客户的业务生存，规模小、网点少、诚信缺失、专业人才严重不足、经营管理水平较低，甚至屡屡出现虚开发票、协助制作假赔案、骗赔、非法挪用保险费等违法违规行为，离“专业化”还相距甚远。种种这些问题，如果不能得到及时有效的解决，必然将动摇专业中介市场的“立业之本”。因此，当前专业中介市场发展的根本问题是树立和提升行业自身的专业化形象。<br/>
<span class="t14">  <span class="t14"> 为了更好地规范中介市场的发展，监管部门可以从以下几个方面入手：（1）继续严格实行市场化制度，通过市场竞争的优胜劣汰机制，实现市场的自动选择功能，让专业化经营水平低下的机构逐步退出市场；（2）充分利用当前中介市场的“阵痛调整期”，鼓励和支持中介公司进行并购与重组，逐步建立一批全国性的专业中介集团，以实现行业资源的优化组合；（3）逐步健全中介机构管理制度，建立和完善中介机构服务行为的专业化标准，规范中介机构的市场竞争行为；（4）及时理顺保险公司与中介机构的关系，既要提升两者的合作水平，又要保持中介机构的独立性和公正性；（5）积极探索高校教育、企业培训和资格考试等多种人才培养模式，建立完善的人才培养、引进和管理体制，提高从业人员的专业素质。</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p>
