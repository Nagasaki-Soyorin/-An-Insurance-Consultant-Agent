<p><span class="t14">  <span class="t14"> <span class="t14"> <span class="t14">目前，医改在中国如火如荼地进行，有关医改的新闻屡屡出现在公众的视野里，许多机构也纷纷成立了研究医改的课题组，提出各自的主张。其中，商业健康保险是否要介入医改，是否要被纳入医疗保障体系是一个焦点问题。有学者认为，商业健康保险具有逐利性，所以不应该被纳入到医疗保障体系中。对于这种观点，笔者并不赞同。<br/>
<span class="t14">  <span class="t14"> <span class="t14"> <span class="t14">与商业健康保险公司一样，公立医院也存在逐利现象。司马迁在《史记?货殖列传》中说道，“天下熙熙，皆为利来；天下攘攘，皆为利往”。司马迁笔下的“天下”，应该包括社会中所有的个人和组织。具有公益性质的公立医院，作为社会中的组织，自然也不能跳出“为利来”、“为利往”的规律。<br/>
<span class="t14">  <span class="t14"> <span class="t14"> <span class="t14">首先，公立医院中的个人存在着逐利现象。这里仅列举几则公开报道。2005年，河北顺平县医院的70多名医生，在两年多的时间里，吃掉药商返给的回扣20多万元。该案涉及医院全部有处方权的科室和90%以上有处方权的医生。2006年，广东省江门市新会区人民医院约200名医生中有140多名陷入了“回扣陷阱”。这样的例子，还有很多很多。分析这些医院集体腐败案，我们可以清晰地看出：从药商、医院院长、药剂科领导到有处方权的医生，一些医院大搞“权钱交易”，形成了多开多得、利益均沾的腐败食物链。<br/>
<span class="t14">  <span class="t14"> <span class="t14"> <span class="t14">可以说，“有人的地方，就有利益的存在”。公立医院从门诊到住院，从小病到大病，都是由人来运作的，那么，必然存在着各种各样的利益，也就必然存在着对利益的追逐。在资源稀缺的情况下，这种逐利性就体现在对稀缺资源的争夺上。即使是在资源丰富的情况下，由于人的欲望是无限的，资源总是处于相对稀缺的状况。所以，人类的逐利性是永远存在的。<br/>
<span class="t14">  <span class="t14"> <span class="t14"> <span class="t14">其次，公立医院作为一个普通的组织，也存在逐利现象。这方面一个典型的例子是医院私设“小金库”。这里同样列举几则公开的报道。2006年3月29日，审计署发布该年度第1号审计公告，公布了2004年度中央预算执行和其他财政收支审计查出问题的纠正结果。其中，北京十家医院在采购中收取折扣、回扣形成的“小金库”已予以没收和罚款。2009年，江苏省江阴市人民医院利用各种方式“克扣”病人药品来牟取暴利。其中，百分之八十的暴利被纳入医院“小金库”统一核算，经过院领导审核后，财务科给各病区和药剂科按比例发放现金。据悉，这种现象在该医院已持续了近10年。<br/>
<span class="t14">  <span class="t14"> <span class="t14"> <span class="t14">组织的逐利性，其受益对象不是个人，而是组织中的所有成员。因此，组织的逐利性往往带有福利的性质，具有很强的隐蔽性，披上合法，至少是合理的“外衣”。此外，中国历来有法不责众的惯例，所以，组织的逐利规模大，危害也大。并且，作为公益性组织的公立医院由于有一定权力，占据一定的垄断地位，其逐利的后果更严重，更重要的是，造成的社会影响也更恶劣。<br/>
<span class="t14">  <span class="t14"> <span class="t14"> <span class="t14">可见，公立医院无论从微观的个人层面，还是从宏观的组织层面，都具有逐利性。但是，从没有人提出要把公立医院排除在医疗保障体系之外。所以，仅仅出于逐利性的缘故而将商业健康保险公司排除在医疗保障体系之外，其理由很不充分。<br/>
<span class="t14">  <span class="t14"> <span class="t14"> <span class="t14">逐利未必是坏事，相反，逐利有时也能增进社会福利。关于这一点，亚当?斯密曾经有个很有名的比喻。他说一个面包师傅之所以要做面包，不是为了让我们吃到好面包，不是为了要填饱我们的肚子，而是因为他要挣钱。很奇妙的是，假如每个人都像这个面包师傅这么自私自利的话，整个社会都会获利。同样，商业健康保险公司出于对自身利益的追求，不断地改善经营管理，降低经营成本，提供各种险种来满足客户的需求的结果，不仅能够促进医疗保障体系的完善，也能够给整个社会增加福利。在医疗保障领域，与公立医院相比，商业健康保险公司起到了竞争、补缺与榜样的作用。<br/>
<span class="t14">  <span class="t14"> <span class="t14"> <span class="t14">商业健康保险公司可以充当“竞争者”的角色。如果公立医院与商业健康保险公司同处在医疗保障体系之内，公众自然就会对二者就服务的质量、价格、品种进行对比。同时，由于处在激烈竞争的市场当中，商业健康保险公司出于自身逐利的考虑，危机感普遍比公立医院强，客观地说，必须比公立医院做得好才能更好地生存。这就对公立医院形成了极大的竞争压力，促使公立医院也努力提升自身服务的质量。<br/>
<span class="t14">  <span class="t14"> <span class="t14"> <span class="t14">商业健康保险公司还可以充当“补缺者”的角色，完善补充医疗保险市场，在基本医疗保障领域之外，及时开发与基本医疗保障互为补充的健康保险产品，满足人民群众更高的保障需求。而对于基本医疗保障没有覆盖到的人群，商业健康保险能够提供适应的产品，满足其保障需求。<br/>
<span class="t14">  <span class="t14">  <span class="t14">此外，商业健康保险公司可以充当“领先者”的角色。商业健康保险公司出于自身生存的考虑，对各种各样的信息变化异常敏感，冒险精神相对较强，乐于、勇于、善于尝试新的技术和设备。而公立医院由于处于一定的垄断地位，生存的压力不是很大，对新技术、新设备的敏感程度远远地低于商业健康保险公司。在逐利性的驱动下，商业健康保险公司率先进行变革，其成功的经验，包括开发的产品、提供的服务都可以为公立医院所借鉴。<br/>
<span class="t14">  <span class="t14">  <span class="t14">昔日，尧在位的时候，黄河流域发生了很大的水灾。尧命令鲧治理洪水。鲧采取水来土掩，筑堤造坝的方式对洪水进行围追堵截，花费九年的时间还未把洪水治好。相反，洪水愈演愈烈，冲毁了堤坝。后来舜让鲧的儿子禹来治理洪水。禹用开渠排水、疏通河道的方式，十三年后，把洪水导入大海。同时，河道里的水可以用来灌溉庄稼，百姓过上幸福的生活。<br/>
<span class="t14">  <span class="t14">  <span class="t14">如果把利益比作洪水的话，它也并不可怕。只要我们能够正视利益的存在，尊重合法、合理的利益诉求，并对其进行疏导，也就是说，只要我们用利益机制来规范人们的行为，那么利益不仅不会造成危害，还会带来更大、更多的益处。但是，一旦我们把利益排斥在制度、体系之外，利益就会像洪水冲破堤坝一样，最终摧毁制度与体系，造成莫大的损失。<br/>
<span class="t14">  <span class="t14"> <span class="t14"> <span class="t14">“逐利性”并不可怕！与利益“共舞”，方才是正确的选择！马克思曾经说过，“思想一旦离开利益，就一定会使自己出丑。”作为思想激烈碰撞的产物，医疗保障体系一旦离开了利益，是不是也会使自己出丑呢？</span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></span></p>
